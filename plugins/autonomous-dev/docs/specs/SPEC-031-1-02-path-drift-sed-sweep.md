## Summary
Apply the deterministic prefix substitution `src/portal/` →
`plugins/autonomous-dev-portal/server/` across every SPEC under
`plugins/autonomous-dev/docs/specs/`, then verify mechanically that no
`src/portal/` substring remains. This spec performs the actual file edits;
it does not populate the matrix or commit (those are handled by
SPEC-031-1-03).

## Functional Requirements

- **FR-1**: A find+sed pipeline MUST apply the prefix substitution
  `s|src/portal/|plugins/autonomous-dev-portal/server/|g` against every
  `*.md` file under `plugins/autonomous-dev/docs/specs/`. Task: PLAN-031-1
  task 3.
- **FR-2**: All `*.md.bak` files generated by the BSD-compatible `sed -i.bak`
  invocation MUST be deleted from the working tree before the spec is
  considered complete. Task: PLAN-031-1 task 3 acceptance criterion.
- **FR-3**: After the substitution, `grep -rln "src/portal/" plugins/autonomous-dev/docs/specs/`
  MUST return zero matches. This is the necessary-but-not-sufficient gate.
- **FR-4**: `git diff --stat plugins/autonomous-dev/docs/specs/` MUST show
  the modified-file count matching the audit list from SPEC-031-1-01. A
  mismatch (any SPEC amended that was not audited; any audited SPEC not
  amended) is a hard failure of this spec.

## Non-Functional Requirements

| Requirement | Target | Measurement Method |
|-------------|--------|---------------------|
| Substitution determinism | Re-running sed on the post-sweep tree produces no further changes | Run sed twice; second invocation's `git diff` must be empty |
| No collateral damage | No file outside `plugins/autonomous-dev/docs/specs/` is modified | `git diff --name-only` shows only paths under that directory |
| BSD/GNU portability | Command works identically on macOS (BSD sed) and Linux (GNU sed) | Use `-i.bak` form (works on both); test on macOS during execution |
| Total runtime | < 30 seconds on a clean working tree | Wall-clock measurement |

## Patterns to Find/Replace

**Find pattern** (literal substring with trailing slash):
```
src/portal/
```

**Replace with** (literal):
```
plugins/autonomous-dev-portal/server/
```

**Sed expression**:
```
s|src/portal/|plugins/autonomous-dev-portal/server/|g
```

The `|` delimiter avoids escaping the embedded `/` characters.

## Verification Commands

```bash
# Apply the sweep (BSD-compatible; -i.bak then remove .bak)
find plugins/autonomous-dev/docs/specs -name "*.md" -exec \
  sed -i.bak 's|src/portal/|plugins/autonomous-dev-portal/server/|g' {} \;
find plugins/autonomous-dev/docs/specs -name "*.md.bak" -delete

# Verify no .md.bak files remain
find plugins/autonomous-dev/docs/specs -name "*.md.bak" | wc -l   # must be 0

# Verify no src/portal/ substring remains
grep -rln "src/portal/" plugins/autonomous-dev/docs/specs/   # must produce no output (exit 1)
echo $?   # must be 1 (grep's "no matches found")

# Verify the modified-file count matches the audit
git diff --name-only plugins/autonomous-dev/docs/specs/ | wc -l   # must equal SPEC-031-1-01 audit count

# Verify no out-of-scope files were modified
git diff --name-only | grep -v "^plugins/autonomous-dev/docs/specs/" | wc -l   # must be 0
```

## Implementation Notes

- The substitution is a **strict prefix replacement**. It maps
  `src/portal/auth/foo.ts` to
  `plugins/autonomous-dev-portal/server/auth/foo.ts` 1:1; subpaths are
  preserved verbatim.
- **What this misses** (intentional; deferred):
  - `src/portal` without trailing slash. TDD OQ-31-01 covers this in a
    follow-up pass; PLAN-031-4's verification script greps for residual
    occurrences and surfaces them.
  - Other path-drift forms (e.g., `./src/portal`, `\`src/portal\``).
    Same disposition.
- **What could go wrong**:
  - The substituted path's target file was renamed (not just relocated)
    during the portal extraction. The substituted text *looks* right but
    references a non-existent file. PLAN-031-4's verification script
    check (4) catches these mechanically. Per TDD OQ-31-03, the SPEC is
    not invented or deleted — the row gets an Open Question note in the
    matrix (recorded by SPEC-031-1-03).
